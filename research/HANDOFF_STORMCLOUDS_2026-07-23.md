1. ЧТО ТАКОЕ STORMCLOUDS
================================================================
Браузерная симуляция «гроза кода» (ваниль HTML/CSS/JS, ноль зависимостей).
Тучи-агенты несут фрагменты Python-подмножества; при столкновении сбрасывают
их в буфер; по «молнии» собранная программа ПО-НАСТОЯЩЕМУ исполняется
встроенным мини-интерпретатором. Тумблеры: память неба, покрытие, эволюция.
Каждая строка в журнале помечена тучей-источником. Проект English-only,
лицензия — 7% роялти (файл LICENSE в репо).
КЛЮЧЕВАЯ НАХОДКА: наивная функция приспособленности в «эволюции» вырождает
популяцию (система оптимизирует «выживание», полезный вывод падает почти к 0);
починка (награда за печать) возвращает вывод. Это ЖИВАЯ иллюстрация reward
hacking / закона Гудхарта — и это стало основой продукта.

4. ПРОДУКТОВЫЙ КУРС — ИТОГ ДВУХ КОНВЕЙЕРНЫХ ПРОГОНОВ
================================================================
ПРОГОН #1 (applications, 5 семейств: Kimi, Grok, Qwen, GPT, Gemini):
  - Сфера = ОБРАЗОВАНИЕ/ДЕМО, НЕ игра/IDE/генератор кода (единогласно).
  - «Голый симулятор ≈ ноль коммерческой ценности — всё в ОБЁРТКЕ» (единогласно).
  - Сильнейшее направление = reward-hacking / «метрики опасны» демо-кит.
  - Первый шаг = упаковка + валидация, НЕ новые фичи.
  - 7% роялти на голый движок = токсично; нужен бесплатный академ-слой.

ПРОГОН #2 (AI-market deep dive, 3 семейства: Kimi, GPT, Qwen):
  - Гипотеза «ИИ везде → большие деньги» = подтверждена НАПОЛОВИНУ: деньги
    реальны, но это НИША-ИНГРЕДИЕНТ в корп. рынке обучения, НЕ крупный
    самостоятельный рынок демо.
  - ПОКУПАТЕЛЬ (единогласно) = корп. L&D / тренеры «ИИ-риски для руководителей»
    / AI-governance консультанты, перепродающие обучение в корпорации.
    НЕ ML-инженеры (слишком игрушечно), НЕ академ AI-safety (гранты).
  - ФОРМА = готовый facilitator ВОРКШОП-КИТ (встраиваемое демо + 30-мин
    методичка + слайды-кейсы + коммерч. лицензия). Голая ссылка не продаётся.
  - НУЖНА non-code «executive» версия (Python на экране отпугивает руководителей):
    тучи=поведения, reward=KPI, полезный вывод=бизнес-результат.
  - ДОКУМЕНТИРУЕМОСТЬ (методичка + learning objective + артефакт прохождения) =
    что отличает платное от бесплатного видео под комплаенс.
  - 7% РОЯЛТИ для B2B = токсично → ФИКС годовая B2B/site-лицензия; роялти
    только для white-label/resale/курс-авторов.
  - ЦЕНЫ: free (демо/академ); ~$100–500/год индивид. тренеру; ~$1 500–3 000/год
    корпорации/консалтингу; white-label дороже.
  - ДРАЙВЕР = корп. AI literacy + EU AI Act Art.4 (надзор с 08.2026). НО закон
    шаткий/мода 1–2 лет → ХЕДЖ на Гудхарта/KPI (вечная тема, шире ИИ).
  - ПЕРВЫЙ ШАГ (единогласно) = НЕ код, а: упаковать демо в 30-мин сценарий +
    провести 3 БЕСПЛАТНЫХ пилота с корп. тренерами → валидировать оплату
    (тест: использовал/заплатил/попросил invoice).

5. СВЯЗЬ С FOUNDATION LAYER (проект автора)
================================================================
StormClouds — интерактивный демонстратор паттерна FO-089 INSTRUMENT_MEASURES_
PROXY (= закон Гудхарта). Поддержка: FO-064, FO-018, FO-004, FF-002.
В методичку продукта уже взяты реальные кейсы из FL CASE_LIBRARY как примеры:
Therac-25, Челленджер, LTCM (мерили одно → хотели другое). Довод продажи —
FO-088 (существование ≠ читаемость ≠ понимание): демо делает паттерн ПОНЯТНЫМ.
FL-аудит (сессия 2026-07-23) нашёл: дубликат FO-014≡FO-096; версия v11.5 vs
v11.3 в одном файле; «покрытие 100%» при ~8 DEFERRED; «~» счётчики при «0
галлюцинаций». Полезность FL для продукта = теоретическая подложка + кейсы.
